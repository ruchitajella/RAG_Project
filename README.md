# 📚 Retrieval-Augmented Generation (RAG) Research Assistant

This project implements a Retrieval-Augmented Generation (RAG) pipeline that allows users to query documents and receive accurate, context-aware answers generated by a Large Language Model (LLM).

The system retrieves relevant information from stored documents using vector embeddings and generates responses using Llama-3 via Groq API.

# 🚀 Project Overview

Traditional LLMs rely only on their pre-trained knowledge, which may lead to outdated or hallucinated answers.
This project solves that problem using RAG (Retrieval-Augmented Generation).

RAG works by:

Retrieving relevant document chunks from a knowledge base.

Supplying that context to the LLM.

Generating accurate answers grounded in the retrieved information.
